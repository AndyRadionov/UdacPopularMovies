# PopularMovies

Android App Shows List of Popular Movies from TMDB

### Used Libraries
 - Retrofit
 - RxAndroid
 - Moxy
 - Butterknife
 - Gson
 - Picasso
 - Justified

### Installation
 - Put your themoviedb.org API key here: res/values/app_settings.xml -> api_key
